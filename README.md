## About Me

I'm a Computer Systems Engineering graduate with hands-on experience in **QA manual testing**, **test automation**, and **AI-powered applications**.

I have worked with **Cypress, JavaScript, TypeScript**, and modern web technologies, and my graduation project was a **RAG-based system** that analyzes Software Requirements Specification documents and generates software features and test cases.

I'm interested in building reliable software, improving testing workflows, and exploring how **Generative AI, RAG, and LLMs** can support software engineering.
