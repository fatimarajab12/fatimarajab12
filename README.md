## About Me

I'm a **Computer Systems Engineering graduate** with hands-on experience in **QA manual testing**, **test automation**, and **AI-powered applications**.

I have experience working with **Cypress, JavaScript, TypeScript, Python**, and modern web technologies. My graduation project focused on building a **RAG-based system** using **LangChain** and **OpenAI APIs** to analyze Software Requirements Specification (SRS) documents and generate software features and test cases.
